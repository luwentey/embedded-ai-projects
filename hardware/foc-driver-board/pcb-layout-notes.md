# FOC 驱动板 — PCB 布局指引

> 项目：FOC 三相无刷电机驱动板 | 层数：4层 | AI 辅助审查

---

## 1. 叠层结构 (4层板)

| 层 | 名称 | 用途 |
|:--:|:----|------|
| L1 | Top | 走线层：功率路径、信号、器件贴装 |
| L2 | GND | 完整地平面（信号地 + 功率地分割） |
| L3 | PWR | 电源平面（VMOTOR、VCC_12V） |
| L4 | Bottom | 走线层：辅助信号、控制信号 |

## 2. 布局要点

### 2.1 功率级

```
┌─────────────────────────────────────────────┐
│       VMOTOR 输入 (P2)                       │
│          │                                    │
│    ┌─────┴─────┐                              │
│    │ 去耦电容   │  ← 100μF+1μF×3              │
│    └─────┬─────┘                              │
│          │                                    │
│    Q1(D) Q3(D) Q5(D)  ← 高端MOSFET漏极        │
│    │      │      │                            │
│    Q1(S)─Q2(D) → U_OUT                        │
│    Q3(S)─Q4(D) → V_OUT                        │
│    Q5(S)─Q6(D) → W_OUT                        │
│    │      │      │                            │
│    Q2(S)  Q4(S)  Q6(S)  ← 经采样电阻到GND     │
└─────────────────────────────────────────────┘
```

### 2.2 关键布局规则

| 规则 | 要求 | 原因 |
|------|:----|------|
| MOSFET 间距 | ≥5mm | 散热间隔，减少热耦合 |
| 采样电阻靠近MOSFET | <10mm | 减少寄生电感影响 |
| 自举电容靠近IC | <5mm | 减少驱动回路寄生电感 |
| 栅极串联电阻靠近MOSFET | <3mm | 抑制栅极振荡 |
| 去耦电容靠近电源引脚 | <3mm | 最小化回路电感 |

### 2.3 走线宽度

| 网络 | 电流 | 建议宽度 | 备注 |
|------|:----:|:--------:|------|
| VMOTOR | 80A | ≥8mm（或铜皮开窗+加锡） | 多层并联 |
| GND_PWR | 80A | ≥8mm | 功率地平面 |
| U_OUT/V_OUT/W_OUT | 80A | ≥8mm | 三相输出 |
| VCC_12V | 500mA | 1mm | - |
| VCC_5V | 5mA | 0.3mm | - |
| 信号线 | 1mA | 0.25mm | 无大电流 |

## 3. 地平面设计

### 3.1 分割策略

```
┌──────────────────────────────────┐
│    信号地 (GND)   │  L2层         │
│    ───── 分割线 ───              │
│    功率地 (GND_PWR) │  L2层        │
└──────────────────────────────────┘
     ↓              ↓
  系统GND        经Rs_BUS
   (P3)       连接至功率地
```

- 信号地 (GND) 与功率地 (GND_PWR) **单点连接**（经 Rs_BUS 底部）
- 所有控制信号参考 GND，功率路径参考 GND_PWR
- L2 层通过分割槽实现两地分离

### 3.2 采样布线注意事项

- 相电流采样差分线（SENSE_U+/U- 等）：等长、平行走线，远离功率开关节点
- 差分对下方保持完整地参考
- Rs_BUS 的 Kelvin 连接：从焊盘内侧引出差分信号，避开大电流路径

## 4. EMI 抑制

| 措施 | 说明 |
|------|------|
| 栅极串联电阻 (10Ω) | 减缓开关沿，降低 di/dt |
| PWM 输入串联电阻 (100Ω) | 限制驱动输入电流 |
| 去耦电容组合 | 大容值+小容值覆盖宽频段 |
| 地平面完整 | 控制回路最小化 |
| 功率回路紧凑 | 每个半桥的MOSFET+采样电阻构成最小回路 |

## 5. AI PCB 审查清单

| 序号 | 审查项 | AI 审查方法 |
|:----:|:-------|:------------|
| 1 | 功率回路面积 | 视觉模型测量开关回路走线长度 |
| 2 | 去耦电容距离 | 测量 IC 到电容焊盘边缘距离 |
| 3 | 采样差分对等长 | 视觉比较差分线长度差 |
| 4 | 散热铜皮面积 | 计算 MOSFET 漏极铜皮面积 |
| 5 | 走线间距 | 检查高低压网络间距 ≥ 0.5mm |
| 6 | 地平面完整性 | 检查 L2 层是否有信号线切断地平面 |
